# Landing Page - The Odin Project

This is a solution to the [Landing Page project](https://www.theodinproject.com/lessons/foundations-landing-page) from The Odin Project's Foundations course.

## Overview

The goal of this project was to recreate a landing page design provided by The Odin Project using HTML and CSS, with a focus on Flexbox for layout.
